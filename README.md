# Vedic Panchanga

GPS-based Vedic Panchanga — Tithi, Nakshatra, Yoga, Karana, Vaar, Muhurtas.

## Deploy to GitHub Pages

1. Create a new GitHub repo named **`vedic-panchanga`**
2. Upload all these files to the repo (main branch)
3. Go to **Settings → Pages → Source** → select **GitHub Actions**
4. The workflow auto-runs on every push — your app will be live at:
   `https://YOUR-USERNAME.github.io/vedic-panchanga/`

> If your repo has a different name, edit `vite.config.ts` line 8:
> `base: "/YOUR-REPO-NAME/",`
